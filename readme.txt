== Description ==

Amazon Product Lister, an extension by CedCommerce allows sellers to feed their Product data from their Woocommerce Store to amazon for connecting it with one of the biggest ecommerce industry players. 

= Key Features =

* Bulk product uploads
* Category mapping with amazon category
* Create and edit profile settings
* Create amazon attributes value only for simple product
* Product data validation
* Profile feature for automating the procedure
* Support simple products upload
* Order listing
* Check Out our Premium Version [Amazon WooCommerce Integration](http://cedcommerce.com/woocommerce-extensions/amazon-woocommerce-integration) for enhanced features.

== Installation ==

= Automatic installation =

In the search field type " Amazon Product Lister" and click Search Plugins. Once you've found our  'Amazon Product Lister' you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading the plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
